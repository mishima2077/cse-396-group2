\needspace{8\baselineskip}
\section{Module Overview}

MOD-04 and MOD-05 form the rover's close-range navigation loop.
MOD-05 performs ultrasonic distance sensing and reports obstacle proximity,
while MOD-04 converts high-level movement requests into low-level differential
drive commands.

These two modules are tightly coupled with the autonomy layer:
MOD-05 continuously feeds distance data to MOD-07, and MOD-07 then commands
MOD-04 to drive, turn, or stop based on mission state and safety checks.

\section{Inter-Module Protocol: MOD-05 to MOD-07 to MOD-04}

\textbf{Step 1 - Sensor Acquisition:}
MOD-05 repeatedly samples range information using \texttt{distance\_read\_cm()}.

\textbf{Step 2 - Data Publishing:}
Each measurement is propagated as distance context for MOD-07 decision logic.

\textbf{Step 3 - Safety Evaluation:}
MOD-07 checks unsafe proximity, either directly through threshold logic or by
using \texttt{distance\_is\_unsafe()} semantics.

\textbf{Step 4 - Command Generation:}
MOD-07 computes a motion command (\texttt{motion\_direction\_t} and speed).

\textbf{Step 5 - Actuation:}
MOD-04 executes the request via \texttt{motion\_drive()} and emits PWM and
direction signals toward MOD-03.

#include "distance_sensing.h"
#include "motion_control.h"

void setup(void) {
    distance_init();
    motion_init();
}

void loop(void) {
    if (distance_is_unsafe()) {
        motion_emergency_stop();
    } else {
        motion_drive(MOTION_DIR_FORWARD, MOTION_DEFAULT_SPEED);
    }
}

\section{Failure Modes and Limitations}

\textbf{Distance sensing limits:}
soft or angled surfaces may absorb or scatter ultrasonic waves, causing
inaccurate distance values.

\textbf{Motion control limits:}
without acceleration ramping and encoder feedback, sudden starts can create
wheel slip and heading drift.

\textbf{Timing sensitivity:}
late processing in MOD-07 may produce stale command execution if distance data
is not consumed in time.

\textbf{Hardware variability:}
battery voltage drops can reduce available motor torque and alter response to a
fixed PWM value.

\needspace{14\baselineskip}
\section{Conclusion}

MOD-05 provides the spatial safety perception needed to avoid collisions, while
MOD-04 provides deterministic actuation for rover movement. Together, they form
the decision-to-motion bridge between MOD-07 autonomy logic and MOD-03
locomotion hardware.
