# CSE 396 Computer Engineering Project - Group 2

The repository for the CSE 396 Computer Engineering Project. This repository contains the source code modules and the final LaTeX report for our project.

## Project Structure
- `CSE_396_HW2/Header files/` - Contains our hardware and software module C/C++ header definitions.
- `CSE_396_HW2/README files/` - Additional documentation for each specific module.
- `CSE_396_HW2/Report/` - Contains the LaTeX source code and images for the project report.

## For Collaborators
1. **Cloning the repository**:
   ```bash
   git clone <your-github-repo-url>
   ```

2. **Branching**:
   - Ccreate a new branch for a feature or a bug fix.
   - E.g., `git checkout -b feature/flame-detection`

3. **Committing Code**:
   - Keep commits small, logical, and write clear commit messages.

## Contribution
Please submit a Pull Request (PR) when your feature is complete, and request a review from at least one teammate before merging into `main`.
